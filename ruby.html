require 'sinatra'

# Route pour la page d'accueil

get '/' do 'Bienvenue sur mon site !' end
# Route pour une page de présentation d'un utilisateur

get '/utilisateur/:id' do "Voici la page de l'utilisateur #{params[:id]} !" end

Cet exemple utilise le framework Sinatra pour créer un serveur web avec deux routes : une pour la page d'accueil, qui renvoie simplement un message de bienvenue, et une pour une page de présentation d'un utilisateur, qui affiche l'identifiant de l'utilisateur passé en paramètre dans l'URL. Pour exécuter cette application, vous pouvez l'enregistrer dans un fichier nommé `app.rb` et exécuter la commande `ruby app.rb`. Ensuite, vous pouvez accéder à l'application à l'URL `http://localhost:4567/` pour la page d'accueil et à l'URL `http://localhost:4567/utilisateur/1` (par exemple) pour la page de présentation de l'utilisateur 1.
